Server-Konfiguration (IONOS VPS)
24.03.2026
🐧
PHP 8.3 + Nginx konfiguriert Server
Ubuntu-Server (sweet-jemison). PHP 8.3-FPM war bereits installiert aber nicht in Nginx eingebunden. Nginx-Konfiguration für vivahome.de erweitert um PHP-FPM FastCGI-Block.
location ~ \.php$ { include snippets/fastcgi-php.conf; fastcgi_pass unix:/run/php/php8.3-fpm.sock; fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name; include fastcgi_params; }
📦
PHPMailer installiert Server
PHPMailer-Repository geklont nach /var/www/vivahome/public/phpmailer/. Dateien liegen in src/: PHPMailer.php, SMTP.php, Exception.php.
/var/www/vivahome/public/phpmailer/src/PHPMailer.php /var/www/vivahome/public/phpmailer/src/SMTP.php /var/www/vivahome/public/phpmailer/src/Exception.php
⚙️
SMTP-Konfiguration (IONOS) Config
Kontaktformular sendet über IONOS SMTP-Server. Zwei Mails pro Anfrage: Eingehende Anfrage an ve@vivahome.de + automatische Bestätigung mit Leistungsübersicht an den Kunden.
Host: smtp.ionos.de · Port: 587 (STARTTLS) Absender: info@vivahome.de Empfänger intern: ve@vivahome.de
